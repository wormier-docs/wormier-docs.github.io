Keybinds

Here are all the key combinations that are provided with the keyboard.

Keybinds related to RGB

  • FN + space = Diplays a palet of color to chose from. Will change backlit color.
  • FN + right CTRL = Change the keyboard backlit effects. There are 18 light effects.
  • FN + left CTRL = Change the side light effect. There are 5 side light effects.
  • FN + Ins = Increase backlit brightness from OFF - 33% - 66% - 100%. Default is 100%.
  • FN + Del = Decreasse backlit brightness from OFF - 33% - 66% - 100%. Default is 100%.

Other keybinds

  • FN + 1-0 = F1-F12
  • FN + up arrow = Page up
  • FN + down arrow = Page down
  • FN + left arrow = Volume up
  • FN + right arrow = Volume down
  • FN + escape = `
  • FN + shift + escape = ~
